## Style Guidelines:

- Background color: Dark navy blue (#191970) to reflect the logo's dark theme.
- Primary color: Electric blue (#7DF9FF) for a tech-focused, vibrant feel. Use this color sparingly, focusing on highlighting key elements of the UI, not as a general fill.
- Accent color: Sky blue (#87CEEB) to complement the primary color and add subtle visual interest.
- Body and headline font: 'Inter' (sans-serif) for a modern, clean, and readable design.
- Use line-based icons in electric blue to maintain consistency with the theme.
- Clean and structured layout with clear sections for easy navigation.
- Subtle transitions and animations to enhance user experience and add a modern touch.
